# Ethan Elite Workout System (Single File)

This project now runs from **one Python file only**: `Main.py`.

## Requirements
- Python 3.11+
- openpyxl

## Run
```bash
python Main.py --output Ethan_Elite_Workout_System.xlsx
```

That command generates the complete workbook with onboarding, program builder, workout logging, progression, body metrics, recovery/sleep, nutrition, weekly review dashboard, help guide, and hidden validation lists.
